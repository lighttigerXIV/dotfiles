# Neovim Config

## Plugins
| Plugin 🔨 | Purpose 🪵 |
|--|--|
| Lazy | Load other plugins |
| Alpha | Provides a splashscreen |
| Auto-Save | It auto saves the files when exiting editing mode |
| Auto-Pairs | When typing `{` it adds `}` and more |
| Catppuccin | The theme 😸 |
| Nvim-LSP | It does completion stuff |
| Lua-Snip | Provides a popup to show snippets and more |
| Nvim-Cmp | Provides snippets and documentation to lua snip |
| Discord-Presence | Shows the files and editor in discord |
| Fterm | Provides a floating terminal |
| Mason | Provides lsp support |
| Lua-Line | Adds a status bar on the bottom |
| Neo-Tree | Adds a file tree |
| None-Ls | Shows functions documentation |
| Barbar | Adds tabs |
| Telescope | Provides a quick way to search files in the project |
| Treesitter | Adds support for formatting |
