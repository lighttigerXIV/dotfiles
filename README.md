# ⚙️ Dotfiles
</div>

Welcome to my humble dotfiles 🤔

Wou will need to install stow for some of the dotfiles:

```bash
sudo pacman -S stow
```

| App      | Install Method                      |
| -------- | ----------------------------------- |
| Bash     | `stow bash`                         |
| Discord  | [Manual](./other-dotfiles/discord/) |
| Firefox  | [Manual](./other-dotfiles/firefox/) |
| Fish     | `stow fish`                         |
| Neovim   | `stow nvim`                         |
| Starship | `stow starship`                     |

# Using stow

To use stow clone this repo into the home directory:
```bash
git clone https://github.com/lighttigerXIV/dotfiles.git ~/dotfiles
```
